pycausalgps

Causal Inference with Generalized Propensity Score

pycausalgps is a Python library for implementing matching on generalized propensity scores (GPS) with continuous exposures. We have developed an innovative approach for estimating causal effects using observational data in settings with continuous exposures and introduced a new framework for GPS caliper matching that jointly matches on both the estimated GPS and exposure levels to fully adjust for confounding bias.
